# Asnowco Website (Static)

## What this is
A modern static site redesign for Asnowco using:
- Asnowco color palette / snowflake branding
- A high-trust, conversion-oriented flow inspired by chrisbuyshomesseattle.com

## Files
- `index.html` Home page
- `blog.html` Publications listing
- `post.html` Publication template
- `css/styles.css` Styles
- `js/main.js` Interactions + publications rendering

## Add your provided images
Copy your image files into `assets/` using these filenames:
- `assets/400pxflake.png`
- `assets/nobgflake.png`
- `assets/banner.png`
- `assets/asnowcologo.png`

## Run locally
Open `index.html` directly in a browser, or use any static server.

## Deploy
Push this folder to a GitHub repository and enable GitHub Pages (or deploy via Netlify/Vercel as a static site).
